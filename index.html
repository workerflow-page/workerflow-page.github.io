workerflow

https://apps.microsoft.com/detail/9pfjph03g2nm?cid=git 
#### **workerFlow**는 바탕화면에 완벽하게 스며드는 투명한 타임라인 달력 위젯입니다.
구글 캘린더와 연동하여 하루의 일정을 직관적으로 파악하고, 나만의 맞춤형 스케줄러로 업무 생산성을 극대화하세요.

WorkerFlow는 Windows 화면 가장자리에 고정되는 타임라인 위젯입니다.
달력을 열고 닫지 않아도, 오늘 하루 전체 일정이 항상 눈앞에 펼쳐집니다.

─────────────────────────────
■ 핵심 기능
─────────────────────────────

[Google 캘린더 완전 연동]
• Google 계정과 OAuth 2.0으로 안전하게 연결
• 일정 자동 동기화 (5분 간격 + 앱 포커스 시 즉시 갱신)
• 앱 내에서 일정 추가 · 수정 · 삭제 — 구글 캘린더에 실시간 반영
• 11가지 Google 캘린더 색상 완벽 지원
• 알림 설정 (10분 / 30분 / 1시간 전), 반복 일정 (매일 / 매주 / 매월 / 매년)

[타임라인 위젯]
• 화면 상단 · 하단 · 좌측 · 우측 — 원하는 위치에 고정
• 드래그로 높이/너비 자유 조절
• 드래그로 로컬 일정 바로 추가
• Google 일정(이탤릭)과 로컬 일정을 한 타임라인에 통합 표시
• 현재 시간 실시간 표시선 (빨간 선)

[월간 달력 뷰]
• 전체 월간 일정을 한눈에 확인
• 날짜 클릭 → 타임라인 위젯 즉시 해당 날짜로 전환
• 더블클릭 → 새 일정 추가 모달 바로 열림
• 드래그로 기간 선택 후 일정 등록

[UI / 커스터마이즈]
• Windows Mica 소재 적용 — 반투명 배경 효과
• 배경 색상 및 투명도 자유 조절
• 폰트 크기 조절 (10px ~ 30px)
• 항상 위에 고정(Pin) 토글
• Windows 시작 시 자동 실행
• 시스템 트레이 상주 — 트레이 아이콘 클릭으로 빠른 표시/숨기기

[다국어 지원]
• 한국어 · English · 日本語 · 中文 · Español
